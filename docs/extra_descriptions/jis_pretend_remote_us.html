from.modifiers.optional = ["any"] にしてあるので、 from.modifiers の詳細度が高い順に記述すること。

例えば、

  1. "@" ( from = open_bracket )
  2. "`" ( from = shift + open_bracket )
の順で書いてしまうと、どちらも "@" にマッチしてしまうので、
  1. "`" ( from = shift + open_bracket )
  2. "@" ( from = open_bracket )
の順に書くこと。

※ from.modifiers.optional = ["any"] にしてあるのは、私がモディファイアを複数同時に使うため ( emacs で C-@ , C-M-% , ... )
