Ergo Fly Keys

Ergonomic anti-RSI key bindings for text editing, loosely based on the US Qwerty versions of ErgoEmacs Mode and Xah Fly Keys.

Features

  • Moves frequently-used keys that are in ergonomically bad spots to better locations.
  • Tuned to feel quick and responsive.
  • Does not trigger accidentally while touch typing.
  • Works in terminal apps, too.

Usage notes

  • This mod is made for US ANSI Qwerty keyboards. If you want to use a different keyboard layout, or another trigger key than space, please download the source code and modify ergoemacs_mode.json.rb
  • For more info on how the keys work in terminal apps, see "Keyboard shortcuts for Terminal apps".
